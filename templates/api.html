Welcome to the Digital Bushman API.

We are providing open access to all of the data recorded during the 18 day expedition in the Okavango Delta.

There are 6 content types available: 'ambit', 'sighting', 'breadcrumb', 'image', 'audio', 'beacon' Each of these content types returns a data file in GeoJSON format. The structure of a call to the API is always the same: http://digitalbushman.org/api/timeline?date=YYYYMMDD&types=TYPE To request multiple days, you can append a 'days' parameter: http://digitalbushman.org/api/timeline?date=YYYYMMDD&types=TYPE&days=# For example, to get all of the animan sightings within two days of September 7th: /api/timeline?date=20130908&types=sighting&days=2
